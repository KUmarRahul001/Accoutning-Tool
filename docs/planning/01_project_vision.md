# Project Vision

## 1. Project Title
Comprehensive GST-Compliant Accounting Web & Mobile Suite (Zoho Books + Tally Prime Hybrid ERP)

## 2. Client
Tinplate Computer Training Center (Jamshedpur, India)

## 3. Executive Summary
The mission is to design, architect, and implement a production-ready enterprise accounting platform tailored for SMEs. The platform will serve as a modern alternative to traditional desktop software by combining the cloud-first, user-friendly approach of Zoho Books with the deep, robust accounting capabilities of Tally Prime.

It will be fully compliant with Indian GST regulations and deliver a seamless experience across web and mobile platforms.

## 4. Problem Statement
Current accounting solutions either lack robust cloud capabilities (Tally Prime) or can be overly generic for specific local compliance and workflows without significant customization (Zoho Books). Tinplate Computer Training Center requires a bespoke, modern ERP system that ensures strict adherence to Indian GST laws while offering enterprise-grade security, offline-capable mobile access, and a scalable architecture for future growth.

## 5. Vision Statement
To empower SMEs in India with a secure, highly scalable, and user-intuitive financial management ecosystem that guarantees GST compliance, streamlines accounting workflows, and provides real-time financial visibility across all devices—whether online or offline.

## 6. Core Pillars
* **Compliance First:** Absolute accuracy in Indian GST calculations, filings, and reporting (CGST, SGST, IGST, HSN/SAC codes).
* **Cloud-Native & Hybrid Access:** Centralized web management paired with an offline-first mobile application for on-the-go billing and expense tracking.
* **Enterprise Security:** Implementing robust authentication (JWT + TOTP), role-based access control, encryption, and comprehensive audit trails.
* **Scalability & Modularity:** A monorepo architecture designed for phased rollout, allowing the MVP to quickly evolve into a full 16-module ERP system.
* **Greenfield Innovation:** Built from scratch using modern technologies (NestJS, React, Flutter, PostgreSQL) to ensure a high-performance, maintainable codebase without legacy technical debt.
